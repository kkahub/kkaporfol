SKILL

  1. HTML

    HTML4.01, HTML5등 DOCTYPE을 이해하고 문서에 맞춰 작업이 가능합니다.

  2. CSS

    css1부터 css3 최신 스타일 까지 다양한 프로퍼티를 알고 SCSS, styled-components 등 사용합니다.

  3. 반응형웹/모바일

    UI/UX를 생각하며 다양한 기기, 브라우저에 최적화된 퍼블리싱을 합니다. 유동적으로 변화하는 반응형웹도 제작이 가능하며 Bootstrap, 워드프레스의 경험이 있습니다.

  4. 웹 표준/접근성

    웹 표준/접근성에 대해 이해하고 다양한 종류와 버전에 따른 브라우저에 맞게 마크업합니다.

  5. Javascript

    함수, 메소드, 클래스, 프로토타입 등 절차지향/객체지향을 이해합니다. 그리고 ES6에 let, const, Arrow Function, async, await, fetch 등 활용합니다.

  6. jQuery

    다양한 메소드를 사용해 기능을 구현합니다. 또한 항상 웹속도와 효율을 고려하며 자바스크립트와 함께 활용하고 있습니다. 플러그인 API 활용, 수정이 가능합니다.

  7. Vue

    babel과 함께 프로젝트에 필요한 Vue 기초 지식이 있습니다. es6-promise polyfill, VeeValidate, vue-paginate 라이브러리경험이 있습니다.

  8. React

    대표적인 SPA 라이브러리로 component를 제작 할 수 있습니다. 또한 redux, redux-toolkit을 이용해 상태관리 합니다.

  9. Next.js

    SEO와 성능 향상 등 강점을 가진 SSR, SSG가 가능한 Next.js로 개발 가능합니다.

  10. Typescript

    명확한 자료 타입을 위해 상황에 맞는 타입 선언, 제네릭 타입, 인터페이스 등을 사용합니다.

  11. 기타

    Git 형상관리툴에 대한 기초스킬(SVN, GitLab, GitHub). webpack, babel, gulp 사용. 액셀 기초
